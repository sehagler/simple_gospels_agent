# simple_gospels_agent

This is a simple RAG agent built using LangChain.  It is based on the simple RAG agent described in:

[Build a RAG agent with LangChain](https://docs.langchain.com/oss/python/langchain/rag)

It uses texts of the Gospels from the King James Version of the Bible obtained from:

[Project Gutenberg](https://www.gutenberg.org/)
